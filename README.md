# Advanced-Internet-Communication
Assignments and Laboratories for 4th year Advanced Internet Communication course

**Lab_1:**
A simple simulation of Internet of Everything; Created a server and a client python programs, so the client (device) is able to enter some simple commands (CONNECT, ADD, REMOVE, READ-value, WRITE-target) to communicate with the server.

**Lab_3:**
A p2p (peer to peer) network application, which is an improvement over lab 1 assignment, the single application consistes of 
client and server, and threading is used in this application.
